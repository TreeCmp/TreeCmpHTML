Download TreeCmp

TreeCmp is a command line tool written in pure Java, can be download as a zip archive. The following versions of library are available:

  • TreeCmp_v1.0-b291.zip - requires Java 1.5 (aka Java 5),
  • TreeCmp_v1.1-b308.zip - contains implementation of a new metric - the Matching Pair (MP) distance, requires Java 1.6 (aka Java 6),
  • TreeCmp_v2.0-b11.zip - contains implementation of a new metric - the MAST (MT), UMAST (UM) and Cophenetic (CO) distance, requires Java 1.7 (aka Java 7).

TreeCmp is provided together with a short manual TreeCmp_manual.pdf. In order to unpack the file any software supporting zip compression, for example free software 7-zip (http://www.7-zip.org/), can be used.

The WEB/Standalone version of application based on Spring framework and can be download as:

  • TreeCmpWEB_v2.0-b41.war - WEB application contains GUI, requires Java 1.8 (aka Java 8),
  • TreeCmpWEB_v2.0-b41.jar - standalone version of WEB application, requires Java 1.8 (aka Java 8)

For more details about the MS and MC metrics see related pages:

  • Matching Split metric
  • Matching Cluster metric
